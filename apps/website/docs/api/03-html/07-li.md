# \<html.li>

<p className="text-xl">The `<html.li>` component lets you render a list item.</p>

## Overview

To display a list item, render the `<html.li>` component.

```jsx
import { html } from 'react-strict-dom';

const Foo = () => (
  <html.ul>
    <html.li />
  </html.ul>
);
```
